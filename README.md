## What problem does this project solve?

Banks and financial institutions need to identify customers who are likely to
default on loan or credit card payments.

This project helps in:
- Understanding customer risk
- Predicting probability of default
- Supporting better credit risk decisions

---

## What I did in this project

- Cleaned and prepared real-world financial data
- Performed feature engineering based on billing and payment history
- Trained a Random Forest classification model
- Evaluated the model using ROC-AUC score
- Built a Streamlit web application
- Deployed the complete system online
